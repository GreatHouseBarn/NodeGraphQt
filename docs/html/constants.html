Constants¶

The NodeGraphQt.constants namespace contains variables and enums used throughout the NodeGraphQt library.
NodeGraphQt.constants.NODE_LAYOUT_VERTICAL = 0¶

Mode for vertical node layout.

NodeGraphQt.constants.NODE_LAYOUT_HORIZONTAL = 1¶

Mode for horizontal node layout.
